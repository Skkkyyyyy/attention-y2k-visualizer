# 🎧 Attention — Y2K Music Visualizer

A nostalgic **Y2K / Windows XP inspired music web experience** built with React + Vite.  
This project reimagines *NewJeans — Attention* as an interactive desktop-style interface.


## 🌐Website
[attention-y2k-visualizer.vercel.app](https://attention-y2k-visualizer.vercel.app)

---

## ✨ Features

- 💿 Desktop-style UI  
- 🎵 Custom audio player (play / pause toggle)  
- 🖼️ Image / media viewer (like old media player)  
- 📝 Notepad-style text window  
- 📸 Retro “Instagram.exe” viewer  
- 🎨 Y2K aesthetic (pixel fonts, glow text, glass effects)  
- 💚 Custom green glossy typography (Windows Media Player inspired)

## 🧠 Concept

This project explores:

- Visualizing music through UI design  
- Recreating early 2000s software aesthetics  
- Turning a song into an interactive digital environment  

Inspired by:

- Windows XP / Windows Media Player  
- Early internet & Web 1.0 interfaces  
- Y2K design language
- NewJeans aesthetic :)
